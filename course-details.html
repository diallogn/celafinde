Web 101

Qu'est ce qu'un site web ? Comment fonctionne t-il ?

Avant de Commencez à coder, nous allons répondre à ces questions pour faire découvrir aux étudiants l'univers dans le quelle ils embarques.

Les bases du HTML

Langage de balises pour l'hypertexte

HTML signifie « HyperText Markup Language » qu'on peut traduire par « langage de balises pour l'hypertexte ». Il est utilisé afin de créer et de représenter le contenu d'une page web et sa structure. D'autres technologies sont utilisées avec HTML pour décrire la présentation d'une page (CSS) et/ou ses fonctionnalités interactives (Javascript).

Les bases du CSS

Feuilles de style en cascade

Les CSS (Cascading Style Sheets en anglais, ou « feuilles de style en cascade ») sont le code utilisé pour mettre en forme une page web. Nous répondrons à des questions comme : Comment rendre mon texte rouge ou noir ? Comment faire apparaître mon contenu à tel endroit de l'écran ? Comment décorer ma page web avec une image ou une couleur d'arrière-plan ?

Les Composants web

Éléments d'interface utilisateur

Maintenant que vous connaissez les bases de HTML et CSS, allons un peu plus loin dans la conception Web et les techniques CSS. Un composant est un élément d'interface utilisateur standard qui a un objectif précis.

Bootstrap

Framework css.

Bootstrap est une collection d'outils utiles à la création du design (graphisme, animation et interactions avec la page dans le navigateur, etc.) de sites et d'applications web. C'est un ensemble qui contient des codes HTML et CSS, des formulaires, boutons, outils de navigation et autres éléments interactifs, ainsi que des extensions JavaScript en option.
